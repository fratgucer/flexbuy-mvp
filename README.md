Second-Hand Marketplace Platform

A modern second-hand marketplace where users can browse, purchase, and manage pre-owned items with flexible payment options such as Klarna and Affirm.

Overview

This project is an e-commerce web platform focused on second-hand product sales. Users can explore listed items, view product details, add products to cart, and complete purchases using supported payment providers. The goal is to provide a simple, secure, and user-friendly resale shopping experience.

Features
	•	Browse second-hand products by category
	•	View detailed product pages
	•	Search and filter listings
	•	Add items to cart
	•	Secure checkout flow
	•	Payment integration with Klarna, Affirm, and other providers
	•	User authentication and account management
	•	Order tracking and purchase history
	•	Responsive design for desktop and mobile

Tech Stack
	•	Frontend: React / Next.js
	•	Backend: Node.js / Express
	•	Database: MongoDB / PostgreSQL
	•	Authentication: JWT / OAuth
	•	Payments: Klarna, Affirm
	•	Styling: Tailwind CSS / CSS Modules

Installation

git clone https://github.com/your-username/your-repo-name.git
cd your-repo-name
npm install
npm run dev

Environment Variables

Create a .env file and add the required environment variables:

DATABASE_URL=your_database_url
JWT_SECRET=your_jwt_secret
KLARNA_API_KEY=your_klarna_key
AFFIRM_PUBLIC_KEY=your_affirm_key
AFFIRM_PRIVATE_KEY=your_affirm_secret

Payment Integration

This project supports flexible checkout options through third-party payment providers such as Klarna and Affirm. Payment processing is handled securely through their official APIs.
